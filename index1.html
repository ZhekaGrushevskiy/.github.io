    УХОДЫ
  • Стоп – акне
      Этапы:
    1. Очищающее молочко(падавляет уже на этапе умывания рост пропиопбактерий акне до 50%).
    2. Топизация(тоник для жирной и проблемной кожи с экстрактом гамамелиса и аллатоином).
    3. Нанесение нормализующего концентрата–антисептика с пчелиным маточным молочком,подавляющего развитие патогенной флоры.
    4. Нанесение очищающей маски–антисептика с коллоидной глиной и ментолом.
    5. Дарсовализация по подсохшей маске(мощное бактерицидное и бактериостатическое воздействие,восстанавливает кровообращение).
    6. Завершающий уход с кремом себобаланс с маслом лесного ореха, жожоба и скваленом.
  • Матирование кожи
      Уход для устранения жирного блеска на лице и нормализации работы сальных желез.
    1. Гель для проблемной кожи с экстрактом гамамелиса, аминокислотами и витамином С.
    2. Тоник для жирной кожи с экстрактом гамамелиса и аллатоином.
    3. Энзимный пилинг(мягкое атравматическое глубокое очищение с экстрактом папайи, энанаса и салициловой кислотой.
    4. Очищающая маска (по типу кожи).
    5. Завершающий уход с матирующим кремом с молочными пептидами.
  • Стоп – стресс
      Успокаивающий уход для чувствительной и раздраженной кожи.
    1. Гидрофильное масло(масла макадамин, авакадо, сливы и витамин Е).
    2. Дермальный тоник–спрей с алое вера и кальцием.
    3. Глубокое очищение тоником–эксфолиантом с экстрактом гамамелиса и молочной кислотой.
    4. Успокаивающая маска с экстрактом мальвы, алое вера и центеллой азиатской.
    5. Завершающий уход с дермальным пептидным кремом.
  • Сияние кожи
      Уход для осветления кожи,выравнивание тона и работа с пигментацией.
    1. Очищающая пенка с АХА–кислотами и маслом камелии.
    2. Дермальный тоник–спрей с алое вера и кальцием.
    3. Глубокое очищение энзимным атравматичным пилингом.
    4. Осветляющая маска с витамином С + фонофорез.
    5. Завершающий уход с сывороткой с витамином С.
  • Анти – эйдж
      Витаминный антивозрастной уход для замедления процессов старения, моментального лифтинга и подтягивания контура лица.
    1. Молочко по типу кожи
    2. Дермальный тоник–спрей с алое вера и кальцием.
    3. Витаминный аха–коктейль.
    4. Маска анти–эйдж по типу кожи.
    5. Завершающий уход с концентратом витамина С и гиалуроновым кремом.
